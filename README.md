# Explora la guitarra | Musicala

Aplicativo web estático para explicar la guitarra a estudiantes principiantes.

## Qué incluye

- Inicio con introducción del instrumento.
- Mapa interactivo de partes de la guitarra.
- Línea de tiempo breve de historia.
- Tipos de guitarra: clásica, acústica y eléctrica.
- Ruta de primeros pasos.
- Acordes básicos interactivos.
- Consejos de cuidado.
- Quiz final con retroalimentación.

## Cómo usar

1. Descomprime el ZIP.
2. Abre `index.html` en el navegador.
3. También puedes subir la carpeta a cualquier hosting estático.

## Importante

No es PWA. No incluye `manifest.json`, `service-worker.js`, instalación offline ni Firebase.
